# Landing Page - The Nice Garden Pinang

Website Landing Page **The Nice Garden Pinang** (Template: **Modern**).

---

## 🚀 Cara Mempublish Landing Page ke GitHub Pages (Gratis)

### Cara 1: Langsung Pakai `index.html` (Tanpa Build Node.js)
1. Buat repository baru di [GitHub](https://github.com/new) (misal: `landing-pinang`).
2. Upload file `index.html` langsung ke repository tersebut.
3. Buka **Settings** -> **Pages** -> Pada **Source**, pilih **Deploy from a branch** -> Branch **main** -> Klik **Save**.
4. Website Anda langsung live di `https://username.github.io/landing-pinang`!

---

### Cara 2: Hosting di Netlify / Vercel / Hostinger / cPanel
- Cukup drag & drop file `index.html` ke panel hosting Anda.
- Tanpa konfig server, langsung aktif!

---

### Cara 3: Edit Kode dengan React + Vite (Untuk Developer)
1. Buka terminal di folder ini.
2. Jalankan `npm install`
3. Jalankan `npm run dev` untuk preview lokal.
4. Jalankan `npm run build` untuk membuat file bundel di folder `dist/`.

---
